# Swiggy Restaurant Market Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on Swiggy restaurant data using Python.

The objective is to analyze:

- Restaurant distribution across cities
- Popular cuisines
- Rating trends
- Pricing patterns
- Offer effectiveness
- Pure Veg vs Non-Veg restaurants

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset Features

- Restaurant Name
- Cuisine
- Rating
- Number of Ratings
- Average Price
- Number of Offers
- Area
- Pure Veg
- Location

## Visualizations

- Top 10 Cities by Restaurant Count
- Top 15 Cuisines
- Rating Distribution
- Price Distribution
- Rating vs Review Count
- Offers vs Rating Analysis
- Pure Veg vs Non-Veg Comparison
- Correlation Heatmap

## Key Insights

- Identified cities with the highest restaurant concentration.
- Discovered the most popular cuisines.
- Analyzed customer ratings and review patterns.
- Investigated relationships between pricing, ratings, and offers.
